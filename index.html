Install and use the package

To use the package in a project:

npm i -D @el3um4s/svelte-titlebar

and then in a file:

import { TitleBar } from " @el3um4s/svelte-titlebar"; <TitleBar {title} {isMaximized} on:clickMinimize={() => log('minimize')} on:clickUnmaximize={unmaximize} on:clickMaximize={maximize} on:clickClose={() => log('close')} class={theme} --background-color={backgroundColor} --text-color={textColor} />
