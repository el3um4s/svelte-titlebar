How to use
To import the package in a project:
npm i -D @el3um4s/svelte-titlebar
To use in a file:
<script>

import { SvelteTitlebar } from "@el3um4s/svelte-titlebar";

</script>

<SvelteTitlebar
  • { title }
  • { isMaximized }
  • on:clickMinimize
  • on:clickUnmaximize
  • on:clickMaximize
  • on:clickClose
/>
