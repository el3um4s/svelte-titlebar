SvelteTitlebar

A Titlebar component for Svelte Projects
How to use
To import the package in a project:
npm i -D @el3um4s/svelte-titlebar
To use in a file:
<script>

import { SvelteTitlebar } from "@el3um4s/svelte-titlebar"

</script>
<SvelteTitlebar
  • title
  • isMaximized
> </SvelteTitlebar>
Props
Name Type Default title undefined TitleisMaximized undefined false
Events
on:clickMinimize
on:clickUnmaximize
on:clickMaximize
on:clickClose
